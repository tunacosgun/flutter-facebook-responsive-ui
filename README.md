## ✨ Features

- **Responsive Viewports**: Adapts elegantly across Mobile, Tablet, and Desktop web views using dynamic sizing and layout builders.
- **Complex UI Trees**: Implementing nested lists, horizontal story carousels, slivers, and rich interactive feed posts.
- **Platform Parity**: Emulates exact padding, colors, shadows, and typography identical to the real-world application.
- **Silky Smooth 60FPS**: Leveraging Flutter's Skia rendering engine to push heavy graphical loads effortlessly.

## 🚀 Getting Started

To explore the responsive behavior of this UI kit on your local machine:

### Prerequisites
- Flutter SDK installed
- iOS/Android Simulator or a Web Browser

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/tunacosgun/flutter-facebook-responsive-ui.git
cd flutter-facebook-responsive-ui

# 2. Get dependencies
flutter pub get

# 3. Run the project
flutter run -d chrome # To test web responsiveness
# or 
flutter run # To test on mobile simulator
```
